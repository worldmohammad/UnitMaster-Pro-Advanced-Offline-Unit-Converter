## ✨ Features

- **📐 12+ Unit Categories:** Length, Mass, Time, Temperature, Pressure, Energy, Power, Speed, Volume, Area, Angle, Digital Data, Currency
- **🔢 Full SI Prefix Support:** Nano (n), Micro (μ), Milli (m), Kilo (k), Mega (M), Giga (G), Tera (T)
- **🌡️ Special Conversions:** Temperature (Celsius, Fahrenheit, Kelvin), Currency (with live API)
- **⭐ Favorites:** Save frequently used conversions
- **📜 History:** Track all past conversions with timestamps
- **🔍 Smart Search:** Find any unit by name, symbol, or category
- **🎨 Dark/Light Theme:** Full theme support with toggle
- **⚙️ Customizable Precision:** Adjustable decimal places and number formats
- **💾 Backup & Restore:** Export/import all data as JSON
- **🔒 100% Offline:** All data stored in IndexedDB—never leaves your browser
